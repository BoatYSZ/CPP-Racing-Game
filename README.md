1，介绍：
这是一个赛车小游戏，使用的语言是C++，API有用来渲染的SDL2。

2, 运行指南：
使用Visual Studio打开CourseworkBase2023_vs2019.sln。设置配置为"Debug"或"Release"，平台为"Win32"，即可运行或调试。

3，键鼠设置：
方向键 - 转动相机方向
WASD/ - 控制水平方向的移动
Ctrl-下降
Space-上升
F-放置/回收油灯
Q - 增宽视野角度
E - 缩窄视野角度

4，值得关注的地方：
PBR的渲染效果主要体现在油灯和路灯上，因为它们的材质拥有齐全的金属度贴图，环境光遮蔽贴图，和法线贴图等。模型文件存放在Graphics\Assessment2\objs\
